## Setup the project

### 1. Virtuelle Umgebung im Ordner ".venv" erstellen
python -m venv .venv

### 2. Virtuelle Umgebung aktivieren
linux/macOS:
source .venv/bin/activate
Windows  - CMD
.venv\Scripts\activate

### 3. Pakete aus requirements.txt installieren
pip install -r requirements.txt
